# HackB0x v2.0

HackBox is the combination of awesome tools and techniques.

## Usage

```bash
git clone https://github.com/samhaxr/hackbox && cd hackbox

virtualenv venv && source venv/bin/activate

pip install -r requirements.txt

python hackbox.py
```
